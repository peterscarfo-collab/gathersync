# GatherSync Testing Rollout Checklist

**For Event Organizer**

Use this checklist to ensure a smooth testing session with your groups.

---

## Phase 1: Build & Prepare (30 minutes)

### Build the APK
- [ ] Click "Build APK" in Manus platform
- [ ] Wait 10-15 minutes for build to complete
- [ ] Download the APK file when ready
- [ ] Test install on your own Android device first

### Upload APK for Distribution
- [ ] Upload APK to Google Drive, Dropbox, or file sharing service
- [ ] Set sharing permissions to "Anyone with link can view"
- [ ] Copy the shareable link
- [ ] Test the link in an incognito browser window

### Prepare Documentation
- [ ] Copy `APK_INSTALLATION_GUIDE.md` content
- [ ] Copy `QUICK_START_CARD.md` content
- [ ] Customize with your name and contact info
- [ ] Save as PDF or share as Google Doc

### Get QR Code for iOS Users
- [ ] Open Manus platform and navigate to GatherSync project
- [ ] In the Preview panel (right side), find the QR code
- [ ] Take a screenshot of the QR code OR copy the Expo link
- [ ] Save for distribution to iOS users
- [ ] Test the QR code with your own iPhone using Expo Go

---

## Phase 2: Create Events (10 minutes)

### Set Up Your 3 Testing Events
- [ ] Open GatherSync on your phone
- [ ] Create Event 1: "GatherSync Testing - Today 2 PM"
  - Event Type: Fixed
  - Date: Today
  - Time: 2:00 PM
  - Add all participants
- [ ] Create Event 2: "GatherSync Testing - Tomorrow 10 AM"
  - Event Type: Fixed
  - Date: Tomorrow
  - Time: 10:00 AM
  - Add same participants
- [ ] Create Event 3: "GatherSync Testing - Tomorrow 2 PM"
  - Event Type: Fixed
  - Date: Tomorrow
  - Time: 2:00 PM
  - Add same participants

### Verify Events
- [ ] All 3 events appear in chronological order
- [ ] All participants listed correctly
- [ ] Dates and times are correct
- [ ] Your RSVP is set (mark yourself as Attending for your preferred time)

---

## Phase 4: Collect RSVPs (24-48 hours)

### Monitor Responses
- [ ] Check each event regularly for new RSVPs
- [ ] Note which time slot has the most "Attending" responses
- [ ] Follow up with "No Response" participants after 12 hours

### Reminder Message Template (if needed):

```
Quick reminder: Please open GatherSync and RSVP to the 3 testing events so we can find the best meeting time. Takes 1 minute!

Haven't installed yet? Here's the link: [APK LINK]

Thanks!
```

---

## Phase 5: Schedule & Conduct Meeting (1 hour)

### Pick the Best Time
- [ ] Review RSVP counts for all 3 events
- [ ] Select the time with most "Attending" responses
- [ ] If tied, pick the earliest available time

### Send Meeting Invitation
- [ ] Create Zoom meeting for the selected time
- [ ] Send calendar invite with Zoom link
- [ ] Include agenda: App demo (10 min) + Feedback discussion (20 min)

### Prepare for Meeting
- [ ] Test screen sharing (show the app on your phone)
- [ ] Prepare questions for feedback:
  - Was installation easy?
  - Was RSVP process clear?
  - What features would you like to see?
  - Would you use this for your own groups?
- [ ] Set up recording (ask permission first)

### During Meeting
- [ ] Record the session
- [ ] Take notes on feedback
- [ ] Ask participants to demonstrate any issues they encountered
- [ ] Collect suggestions for improvements

---

## Phase 6: Post-Testing Follow-Up (30 minutes)

### Thank Participants
- [ ] Send thank-you message
- [ ] Share key takeaways from the session
- [ ] Let them know when the next version will be ready

### Analyze Feedback
- [ ] Review recording and notes
- [ ] Categorize feedback: bugs, feature requests, UX improvements
- [ ] Prioritize top 3-5 items to address

### Plan Next Steps
- [ ] Fix critical bugs
- [ ] Implement top requested features
- [ ] Schedule follow-up testing session (optional)

## Troubleshooting Guide

**Participant can't install APK:**
- Check if they enabled "Unknown Sources"
- Verify the APK link is working
- Ask them to try a different file manager app
- As last resort, offer Expo Go alternative

**Participant can't find events:**
- Ask them to pull down to refresh
- Verify they opened the correct app
- Check if events were created correctly on your end

**App crashes for participant:**
- Ask them to clear app data or reinstall
- Check if their Android version is 8.0 or newer
- Collect crash details for debugging

**Low RSVP rate:**
- Send reminder message after 12 hours
- Offer to help with installation via phone call
- Consider extending RSVP deadline
