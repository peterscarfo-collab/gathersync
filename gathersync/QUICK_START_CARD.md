## What is GatherSync?

An app that helps groups find the best meeting time by letting everyone mark their availability.

---

## Installation (5 minutes)

**Android Users:**
1. Download the APK from the link provided
2. Enable "Install from Unknown Sources" when prompted
3. Tap the downloaded file and install
4. Open GatherSync

**iOS Users (Temporary Testing):**
1. Install **Expo Go** from App Store (free)
2. Scan the QR code provided by organizer
3. GatherSync opens in Expo Go
4. Full iOS app coming later!

---

## How to RSVP (30 seconds)

1. Open GatherSync app
2. Find the event (e.g., "GatherSync Testing - Today 2 PM")
3. Tap your name in the participant list
4. Select **Attending** or **Not Attending**
5. Done! The organizer can see your response
